# Hydrological Thresholds: A Reproducible Geospatial Workflow

This repository documents a geospatial workflow for examining hydrological thresholds, including drainage patterns, watershed structure, and terrain constrictions associated with selected locations preserved in narrative memory.

The workflow was developed in connection with the article “Hydrological Thresholds and Narrative Geography in the Southern Levant” and Appendix A of *We Were Not Looking*. 

> **Methodological Note:** This is an open methodological question, not a validated historical model. The workflow identifies and visualizes environmental variables; it does not propose reconstructed itineraries or claim historical proof.

## Repository Contents
* `/scripts`: Python routine execution for data preparation, terrain analysis, and mapping.
* `/figures`: Exported high-resolution panels alongside dedicated contextual data/limitations.
* `/docs`: Detailed methods note and structural limitations of the workflow.
